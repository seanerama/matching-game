# AI Image Transformation Prompts

These prompts are designed for AI photo editing tools that take a source image and transform it. Upload the child's photo first, then use these prompts to transform them into each profession.

## Recommended AI Tools
- **[Midjourney](https://midjourney.com)** - Use with `--cref` (character reference) flag
- **[DALL-E 3 / ChatGPT](https://chat.openai.com)** - Upload photo and request transformation
- **[Leonardo AI](https://leonardo.ai)** - Use Image Guidance feature
- **[Fotor AI](https://www.fotor.com/features/ai-change-clothes/)** - Clothes/costume changer
- **[LightX](https://www.lightxeditor.com)** - AI costume generator
- **[Photoroom](https://www.photoroom.com)** - Background + outfit changes

## How to Use These Prompts

1. Upload the source photo of childName
2. Copy the prompt for the desired profession
3. Paste and generate
4. Review and regenerate if needed
5. Save as: `childName-profession.png` (e.g., `finley-astronaut.png`)

## Style Notes
- Target style: Realistic with subtle illustrated/animated quality
- Vivid, detailed backgrounds
- Maintain the child's facial features and likeness
- Consistent portrait framing (head and shoulders to waist)

## Tips for Best Results

1. **Source photo quality**: Use a clear, front-facing photo with good lighting
2. **Consistency**: Process all images for one child in the same session if possible
3. **Regenerate if needed**: Don't settle for awkward results - regenerate 2-3 times
4. **Square crop**: Crop final images to square (1:1) ratio for card format
5. **Resolution**: Aim for at least 512x512 pixels, ideally 1024x1024
